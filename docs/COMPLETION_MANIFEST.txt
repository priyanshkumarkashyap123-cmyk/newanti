================================================================================
                              QUICK START COMMANDS
================================================================================

FOR LOCAL DEVELOPMENT:
  1. python config_manager.py local
  2. cd apps/backend-python && python -m venv venv && source venv/bin/activate
  3. pip install -r requirements.txt
  4. python -m uvicorn main:app --reload --port 8000
  5. [NEW TERMINAL] cd apps/web && pnpm install && pnpm run dev
  6. ./verify-system.sh

FOR AZURE PRODUCTION:
  1. python config_manager.py azure \
       --resource-group beamlab-ci-rg \
       --app-name beamlab-backend-python \
       --api-key "AIza_YOUR_ACTUAL_KEY"
  2. cd apps/web && pnpm run build
  3. Deploy dist/ to Azure Static Web App
  4. python config_manager.py verify-azure \
       --resource-group beamlab-ci-rg \
       --app-name beamlab-backend-python

VERIFY ANYTIME:
  ./verify-system.sh
  # or for Azure:
  ./verify-system.sh https://beamlab-backend-python.azurewebsites.net

                           ARCHITECTURE SUMMARY

FRONTEND (React/TypeScript/Vite)
  ├─ Browser loads app
  ├─ Check: navigator.gpu exists?
  ├─ YES → Load Rust/WASM solver
  │   └─ Offload heavy calculations to user GPU
  └─ NO → Use Python backend API
      └─ Send requests to FastAPI server

PYTHON BACKEND (FastAPI)
  ├─ Load env with intelligent fallbacks
  ├─ Initialize CORS (beamlabultimate.tech + localhost)
  ├─ Start AI engine (Gemini if key+enabled, else Mock)
  ├─ Health endpoints (/, /health, /ai/status)
  └─ Analysis endpoints (/ai/diagnose, /fix, /modify)

AI ENGINE
  ├─ Production: Google Gemini API
  │   └─ Cost: ~$0.001 per call
  └─ Development: Mock responses
      └─ Cost: Free, instant responses

================================================================================
                         ENVIRONMENT VARIABLES
================================================================================

REQUIRED (with smart defaults):

  GEMINI_API_KEY
    • Local default: mock-key-local-dev
    • Production: Your actual Gemini API key
    • Set via: config_manager.py or Azure CLI

  USE_MOCK_AI
    • Local default: true (development mode)
    • Production: false (use real Gemini API)
    • Set via: config_manager.py or Azure CLI

  FRONTEND_URL
    • Local default: http://localhost:5173
    • Production: https://beamlabultimate.tech
    • Set via: config_manager.py or Azure CLI

  ALLOWED_ORIGINS (CORS whitelist)
    • Local default: http://localhost:5173,http://localhost:3001,...
    • Production: https://beamlabultimate.tech,...
    • Set via: config_manager.py or Azure CLI

HOW TO SET:

  Option 1 - Python Config Manager (Recommended):
    python config_manager.py local          # Interactive setup
    python config_manager.py azure ...      # Azure setup

  Option 2 - Create .env file manually:
    Copy .env.example to .env
    Edit values as needed

  Option 3 - Azure portal:
    App Service → Configuration → Application Settings

  Option 4 - Azure CLI:
    az webapp config appsettings set ...
