```
Step 1: Highlight the toolbar area

┌───────────────────────────────────────────────────────────────────┐
│                                                                   │
│  ┌════════════════════════════════════════════════┐← highlighted │
│  ║ [Geometry] [Properties] [Loading] [Analysis]  ║  pulsing     │
│  ║                                                ║  blue border │
│  ╠════════════════════════════════════════════════╣              │
│  ║ [Node] [Member] [Grid] [Wizard] [Select]      ║              │
│  ╚════════════════════════════════════════════════╝              │
│                                                                   │
│  ┌──────────────────────────────────────────┐                    │
│  │ 📐 This is the Ribbon Toolbar          │  ← tooltip         │
│  │                                          │                    │
│  │ Switch between workflow tabs to access   │                    │
│  │ different tools for modeling, loading,   │                    │
│  │ analysis, and design.                    │                    │
│  │                                          │                    │
│  │ Try clicking "Geometry" to start.       │                    │
│  │                                          │                    │
│  │ Step 1 of 8      [← Back] [Next →] [✕] │                    │
│  └──────────────────────────────────────────┘                    │
│                                                                   │
│  Rest of screen dimmed with dark overlay (#000 @ 50%)            │
│                                                                   │
└───────────────────────────────────────────────────────────────────┘

Tour stops (8 steps):
  1. Ribbon Toolbar — workflow tabs overview
  2. Geometry tab — node and member creation
  3. 3D Viewport — navigation (orbit, pan, zoom)
  4. Properties Panel — element properties
  5. Workflow Sidebar — model tree
  6. Loading tab — applying loads
  7. Analysis tab — running analysis
  8. Results — viewing BMD/SFD/deformed shape

Tooltip style:
  Background: white (#ffffff) or dark (#1e293b) depending on theme
  Border: 1px solid slate-300/700
  Shadow: elevation-3
  Arrow: 8px triangle pointing toward highlighted element
  Max width: 320px
  Progress: dots or "Step X of Y"
```
