## ⚡ Quick Start (Choose ONE)

### **Option A: Use Alternate Account (Recommended)**
```bash
# Run the automated setup script
./setup-alternate-github-account.sh

# Then follow the on-screen instructions:
# 1. Transfer repository on GitHub
# 2. Add secrets to new repo
# 3. Test with: git push origin main
```

**Benefits:**
- ✅ 3,000 free minutes/month on alternate account
- ✅ No infrastructure to manage
- ✅ Same workflow performance
- ✅ 10-minute setup

---

### **Option B: Self-Hosted Runner**
```bash
# 1. Install runner on your machine
mkdir ~/actions-runner && cd ~/actions-runner
curl -o actions-runner-osx-x64-2.311.0.tar.gz -L \
  https://github.com/actions/runner/releases/download/v2.311.0/actions-runner-osx-x64-2.311.0.tar.gz
tar xzf ./actions-runner-osx-x64-2.311.0.tar.gz

# 2. Configure (get token from GitHub repo settings)
./config.sh --url https://github.com/YOUR_ACCOUNT/newanti --token YOUR_TOKEN

# 3. Run the runner
./run.sh

# 4. Update workflows to use self-hosted
# Rename *-selfhosted.yml files to replace original workflows
```

**Benefits:**
- ✅ Unlimited minutes (free)
- ✅ Faster builds (your hardware)
- ✅ No monthly limits

**Drawbacks:**
- ⚠️ Requires machine to be online
- ⚠️ You manage the infrastructure
