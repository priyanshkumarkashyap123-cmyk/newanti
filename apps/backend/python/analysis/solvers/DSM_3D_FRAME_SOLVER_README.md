"""\nDSM_3D_FRAME_SOLVER_README.md - Complete documentation for the backend solver\n\nDirect Stiffness Method 3D Frame Solver with Timoshenko Beam Elements\n========================================================================\n\n## What is this?\n\nA production-ready Python implementation of the Direct Stiffness Method (DSM) for analyzing\n3D frame structures with:\n\n- **3D Timoshenko Beam Elements** (12 DOF per element = 6 DOF × 2 nodes)\n- **Coordinate Transformations** from local to global coordinates (3×3 direction cosines → 12×12 T matrix)\n- **Sparse Matrix Assembly** using scipy.sparse (lil_matrix for construction, csr_matrix for solving)\n- **Boundary Conditions** via Penalty Method (K[j,j] += λ for restrained DOFs, λ = 1e14)\n- **Element Force Back-Calculation** from nodal displacements\n- **Load Combinations** per IS 456:2000, IS 800:2007, IS 1893:2016\n- **Code Compliance Checks** against Indian Standards\n- **FastAPI Integration** for REST API access\n\n## Architecture\n\n```\nReact Frontend                          FastAPI Backend\n┌─────────────────────┐                ┌──────────────────────┐\n│ StructuralModeling  │                │                      │\n│ Canvas              │──── JSON ──────▶ api_routes.py        │\n│ (nodes, elements)   │                │ ↓                    │\n└─────────────────────┘                │ DirectStiffness      │\n                                        │ Method3D             │\n                                        │ ├─ build_model()     │\n                                        │ ├─ assemble_system() │\n                                        │ ├─ solve()           │\n                                        │ └─ back_calc_forces()│\n                                        │                      │\n                                        │ Member Force         │\n                                        │ Calculator           │\n                                        │                      │\n                                        │ Load Combinator      │\n                                        │ (IS 456, IS 800)     │\n                                        │                      │\n                                        │ Code Compliance      │\n                                        │ Checker              │\n                                        └──────────────────────┘\n                                                 △\n                                                 │\n                                            JSON response\n                                         (displacements,\n                                          reactions,\n                                          member forces)\n```\n\n## Module Breakdown\n\n### 1. `dsm_3d_frame.py` (920+ lines)\n\nCore Direct Stiffness Method implementation.\n\n**Classes:**\n\n- **`Node3D`** - Node definition with coordinates and boundary conditions\n  - Fields: id, x, y, z (float64), restraints dict\n  - DOF ordering: [Ux, Uy, Uz, Rx, Ry, Rz]\n\n- **`Element3D`** - 3D Timoshenko beam element definition\n  - Fields: id, node_i, node_j, E, G, A, Iy, Iz, J\n  - Shear areas: Ay, Az (configurable, default 0.012 m²)\n  - Releases: dict for hinged/fixed ends\n\n- **`StructuralModel`** - Container for nodes and elements\n  - Computed: n_nodes, n_elements, n_dofs (= n_nodes × 6)\n\n- **`TimoshenkoBeamElement3D`** (static methods)\n  - `local_stiffness_matrix(element, length)` → 12×12 k matrix with shear deformation\n  - Shear parameters: φy = EIy/(GAyL²), φz = EIz/(GAzL²)\n  - All 12 coupling terms included (axial + torsion + bending Y/Z)\n  - `apply_member_releases(k, releases)` → zeros out rows/cols for hinges\n\n- **`CoordinateTransformations3D`** (static methods)\n  - `direction_cosines(node_i, node_j)` → 3×3 matrix\n    * Local x: along member axis (I→J)\n    * Local y: perpendicular to x in XY plane\n    * Local z: right-hand rule (x × y)\n  - `transformation_matrix_12x12(cos_mat)` → 12×12 T matrix\n    * Repeats 3×3 cosine matrix at 4 block positions\n    * T[0:3, 0:3] = T[3:6, 3:6] = cos_mat (node i)\n    * T[6:9, 6:9] = T[9:12, 9:12] = cos_mat (node j)\n  - `transform_stiffness_matrix(k_local, T)` → K_global = T^T @ K_local @ T\n\n- **`GlobalAssembly`** (static method)\n  - `assemble_global_stiffness(model, n_dofs)` → (K_csr, dof_map)\n    * Uses lil_matrix for O(1) element insertion\n    * Maps nodes to indices: sorted(nodes.keys())\n    * For each element:\n      - Generate local k (12×12)\n      - Transform to global K via [T]\n      - Add to sparse matrix at correct DOF indices\n    * Converts lil→csr for efficient solving\n\n- **`BoundaryConditionHandler`** (static method)\n  - `apply_restraints(K, F, model, node_to_index)` → (K_penalized, F_penalized)\n    * Penalty method: K[j,j] += λ (λ = 1e14)\n    * Sets F[j] = 0 for restrained DOF j\n    * No DOF elimination (system size stays consistent)\n\n- **`DirectStiffnessMethod3D`** (orchestrator)\n  - `build_model(model)` - validates, computes n_dofs\n  - `assemble_system(model, F)` - builds K, applies BCs\n  - `solve()` - scipy.sparse.linalg.spsolve([K]{U}={F})\n  - `back_calculate_reactions()` - {R} = [K]{U} - {F}\n  - Stores: displacements (U), reactions (R), element_forces\n\n**Precision:**\n\n- All calculations in float64 (numpy.float64)\n- Parameters cast explicitly: `float64(element.E)`\n- Matrix determinant check: det(T) ≈ 1.0 (validates coordinate transform)\n\n**Validation:**\n\n- Verified against Bathe, Cook finite element textbooks\n- Hand-checked simple cases (cantilever, simply supported beam)\n- Shear deformation correction validated against Timoshenko theory\n\n### 2. `json_handler.py` (350+ lines)\n\nBridge JSON from React frontend to DSM model objects and vice versa.\n\n**Classes:**\n\n- **`JSONHandler`**\n  - `parse_topology_json(json_data)` → StructuralModel\n    * Input: {\"nodes\": [{id, x, y, z}, ...], \"elements\": [{id, startNodeId, endNodeId, ...}, ...]}\n    * Validates node existence for each element\n    * Returns model with n_dofs = n_nodes × 6\n  - `parse_loads_json(json_data)` → F (ndarray, n_dofs)\n    * Parses \"loads\": [{nodeId, Px, Py, Pz, Mx, My, Mz}]\n    * Maps to global DOF vector\n    * Supports member distributed loads (TODO: convert to nodal equivalents)\n\n- **`OutputFormatter`**\n  - `format_results(dsm, model)` → dict\n    * Extracts displacements/reactions from DSM results\n    * Maps back to node IDs: {nodeId: {ux, uy, uz, rx, ry, rz}}\n    * Returns: {status, displacements, reactions, nodes[], elements[]}\n  - `to_json_string(results)` → JSON string\n    * Uses json.dumps() with default=float for serialization\n\n**Example Usage:**\n\n```python\njson_input = {\n    \"nodes\": [\n        {\"id\": \"N1\", \"x\": 0, \"y\": 0, \"z\": 0},\n        {\"id\": \"N2\", \"x\": 5, \"y\": 0, \"z\": 0}\n    ],\n    \"elements\": [\n        {\n            \"id\": \"M1\",\n            \"startNodeId\": \"N1\",\n            \"endNodeId\": \"N2\",\n            \"E\": 30000,\n            \"A\": 0.25,\n            \"Iy\": 0.002\n        }\n    ]\n}\n\nmodel = JSONHandler.parse_topology_json(json_input)\ndsm = DirectStiffnessMethod3D()\ndsm.build_model(model)\ndsm.assemble_system(model, F)\nU = dsm.solve()\nresults = OutputFormatter.format_results(dsm, model)\n```\n\n### 3. `member_forces.py` (400+ lines)\n\nExtract internal forces (axial, shear, moment) from nodal displacements.\n\n**Classes:**\n\n- **`MemberForceCalculator`**\n  - `calculate_member_forces(model, displacements)` → Dict[member_id, forces]\n    * Process:\n      1. Extract local nodal displacements for each element\n      2. Multiply by local stiffness matrix: {f} = [k]{u}\n      3. Extract 12 force components (6 at each end)\n    * Output: {\"M1\": {axial_i, shear_y_i, ..., moment_z_j}, ...}\n  - `get_diagram_data(member_forces, model)` → SFD/BMD/AFD data\n    * Interpolates forces along member length (21 points default)\n    * Returns x_values, shear_y, moment_z, etc. for plotting\n  - `max_forces(member_forces)` → Dict[max_force_type, value]\n    * Finds maximum absolute forces across all members\n    * Useful for design checks and summary reporting\n\n### 4. `load_combinations.py` (350+ lines)\n\nLoad combination per Indian Standards.\n\n**Classes:**\n\n- **`LoadCase`** (dataclass)\n  - name, description, load_vector (ndarray), load_type\n\n- **`LoadCombination`** (dataclass)\n  - name, description, factors (Dict), envelope_type\n\n- **`LoadCombinator`**\n  - `is456_ultimate_combinations()` - IS 456:2000 concrete ULS\n    * 1.5*DL + 1.5*LL\n    * 1.2*DL + 1.2*LL + 1.2*WL\n    * 1.5*DL + 1.5*WL\n    * 0.9*DL + 1.5*WL (tension)\n  - `is456_serviceability_combinations()` - Deflection checks\n    * DL + LL\n    * DL + 0.8*LL\n    * DL\n  - `is800_ultimate_combinations()` - IS 800:2007 steel ULS\n    * 1.5*DL + 1.5*LL\n    * 1.05*DL + 1.5*LL + 0.9*WL (wind assists)\n    * 1.5*DL + 0.9*LL + 1.5*WL (wind opposes)\n    * 0.9*DL + 1.5*WL (uplift)\n  - `generate_combinations(load_cases, code, envelope_type)` → Dict[combo_name, F_combined]\n  - `envelope_analysis(results)` → max/min across combinations\n\n### 5. `code_compliance.py` (400+ lines)\n\nCode compliance checks against Indian Standards.\n\n**Classes:**\n\n- **`ComplianceCheck`** (dataclass)\n  - check_name, pass_fail, value, limit, design_ratio\n  - code_reference, severity, message\n\n- **`IS456Checker`**\n  - `check_concrete_stress(stress_mpa, grade)` → ComplianceCheck\n    * Permissible = 0.33*fck (Clause 39.1)\n    * Grades: M20/M25/M30/M35/M40\n  - `check_deflection(deflection_mm, span_mm, element_type)` → ComplianceCheck\n    * Limits: beam span/240, cantilever span/180\n    * Table 4: IS 456:2000\n\n- **`IS800Checker`**\n  - `check_steel_stress(stress_mpa, grade)` → ComplianceCheck\n    * Permissible = 0.6*fy (Clause 5.7.1)\n    * Grades: Fe250/Fe410/Fe450/Fe550\n  - `check_slenderness(length_mm, radius_gyration_mm)` → ComplianceCheck\n    * Column λ ≤ 200, tension λ ≤ 250\n    * Clause 5.1.1\n  - `check_shear_buckling(height, thickness)` → ComplianceCheck\n    * Web d/t ≤ 85*sqrt(250/fy)\n    * Table 4\n\n- **`CodeCompliance`**\n  - `run_checks(results, code_type, material_grade)` → Dict\n    * Returns: {passed, failed, warnings, info}\n  - `summary(checks)` → human readable report\n\n### 6. `api_routes.py` (400+ lines)\n\nFastAPI REST endpoints for external submission.\n\n**Endpoints:**\n\n- **POST `/api/v1/analysis/structural`**\n  -
